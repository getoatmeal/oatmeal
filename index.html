Oatmeal

A low cholesterol front-end framework
for fast and light web interfaces.

Why we use oatmeal?

Oatmeal can help you to build small or large project with modular components. So, you can add or remove the fat that you want on your style.

Features

Customize

You can customize your build by add,
remove or modify the components.

Rapid Development

Oatmeal come from preprocessors.
So, you can quickly build your projects.

One For All

Build once and your project will
be compatible with all devices.

How to use

Basic Template

Start with this basic HTML template, to begin using oatmeal.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <meta name="description" content="">
    <meta name="author" content="">
    <title>Your Title</title>
    <!-- Oatmeal Style Sheet -->
    <link href="css/oatmeal.css" rel="stylesheet">
    <!-- Your Style Style Sheet -->
    <link href="css/style.css" rel="stylesheet">
    <!--[if lt IE 9]>
    <script src="https://oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
    <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <!-- your codes start here -->
  </body>
</html>

Forms

you only need to add 1 class in form. .form-oatmeal and its done. all form types will have a better style.

Examples of form usages:

<form class="form-oatmeal">
  <label>Name</label>
    <input type="name" placeholder="Write your name"/>
  <label>Email</label>
    <input type="email" placeholder="youremail@domain.com"/>
  <label>Gender</label>
    <input type="radio" name="gender" id="male" checked><label for="male">Male</label>
    <input type="radio" name="gender" id="female"><label for="female">Female</label>
  <label>Habit</label>
    <input type="checkbox" name="habit" id="smoke"><label for="smoke">Smoking</label>
    <input type="checkbox" name="habit" id="drink"><label for="drink">Drinking</label>
  <label>Write Your Story</label>
    <textarea placeholder="write down your story"></textarea>
  <button type="submit">Submit</button>
</form>

see... I've told you... just add one class to fit your style.

Tables
